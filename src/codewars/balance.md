# 簡易支票簿對帳

## 題目說明

你會得到一個（小型）支票簿字串，其中有時會包含一些雜亂的（非字母數字）字元：

```
"1000.00
125 Market 125.45
126 Hardware 34.95
127 Video 7.45
128 Book 14.32
129 Gasoline 16.10"
```

第一行顯示原始餘額。其他每一行（當不是空行時）提供資訊：支票號碼、類別、支票金額。（輸入格式可能會根據程式語言而有所不同）。

首先，你必須清理這些行，只保留字母、數字、點和空格。

然後回傳一個報告字串（底線表示空格——不要把它們放在你的解答中。它們只是為了讓你看清楚需要幾個空格）：

```
"Original_Balance:_1000.00
125_Market_125.45_Balance_874.55
126_Hardware_34.95_Balance_839.60
127_Video_7.45_Balance_832.15
128_Book_14.32_Balance_817.83
129_Gasoline_16.10_Balance_801.73
Total_expense__198.27
Average_expense__39.65"
```

在報告的每一行上，你需要加上新的餘額，然後在最後兩行顯示總支出和平均支出。為了不讓結果字串太長，我們不要求格式化輸出。

#### 注意事項

- 請參考範例測試中的輸入範例。
- 可能會有一行（或多行）是空白的。
- 將計算結果四捨五入到小數點後 2 位（Elm：不含尾隨的 0）
- 結果的行分隔符可能取決於程式語言 `\n` 或 `\r\n`。請參考「範例測試」中的例子。
- R 語言：不要使用 R 的基礎函式 "mean()"，因為可能會產生與預期略有不同的結果。
