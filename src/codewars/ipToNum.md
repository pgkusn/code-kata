# IP 位址轉數字

[IPv4 位址](http://en.wikipedia.org/wiki/IP_address)是一個 32 位元的數字，用於識別網際網路上的設備。

雖然電腦以 32 位元數字來讀寫 IP 位址，但我們更習慣以點分十進制表示法來閱讀它們，這基本上是將數字分成 4 個 8 位元的區塊，轉換為十進制，並用點號分隔。

在這個題目中，你需要建立函式 `ipToNum`（將 IP 位址轉換為數字）以及函式 `numToIp`（將數字轉換為 IP 位址字串）。輸入永遠是有效的。

## 轉換範例

```javascript
// 原始 IP 位址
192.168.1.1

// 分解為 4 個二進制八位元組
11000000 . 10101000 . 00000001 . 00000001

// 合併在一起（無符號 32 位元二進制）
11000000101010000000000100000001

// 最後轉換為十進制
3232235777
```

注意：二進制八位元組是**無符號**的（所以我們不能有負數）。

請特別注意：JavaScript 對**有符號** 32 位元整數進行位元運算。

## 範例

### ipToNum

```
'192.168.1.1' 轉換為 3232235777
'10.0.0.0'    轉換為  167772160
'176.16.0.1'  轉換為 2953838593
```

### numToIp

```
3232235777 轉換為 '192.168.1.1'
 167772160 轉換為    '10.0.0.0'
2953838593 轉換為  '176.16.0.1'
```

## Solution

```ts
export function ipToNum(ip: string): number {
  return 0
}

export function numToIp(x: number): string {
  return '0.0.0.0'
}
```
