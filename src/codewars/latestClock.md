# 最晚的時鐘

撰寫一個函式，接收 4 個數字並回傳這些數字能組成的最晚時間。

時間格式應為 `HH:MM`。

範例：

```
數字: 1, 9, 8, 3 => 結果: "19:38"
數字: 9, 1, 2, 5 => 結果: "21:59" (19:25 也是有效時間，但 21:59 更晚)
```

### 注意事項

- 結果必須是有效的 24 小時制時間，介於 `00:00` 和 `23:59` 之間。
- 只會測試有有效答案的輸入。

## Solution

```ts
export function latestClock(a: number, b: number, c: number, d: number):string {
  return "00:00";
}
```
